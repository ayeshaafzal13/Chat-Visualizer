# Chat-Visualizer
A C++ graphics project that turns data into clear and colourful visual charts, featuring 10 unique chart types.
# Charts Implemented
Bar Chart
Pie Chart
Line Chart
Donut Chart
Scatter Plot
Area Chart
Column Chart
Histogram
Stacked Bar Chart
Multi-Line Chart
# Technologies Used
C++
graphics.h 
Turbo C++ / WinBGIm
Dev C++ 
# How to Run
1. Clone the repository
2. Compile the source code
3. Run the executable
4. Navigate through the menu to visualize chat data
